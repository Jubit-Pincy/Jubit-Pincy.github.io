International Cuisine

Workshop Assignment

As part of a workshop, our class was asked to recreate a given web design. At the time we were only beginning to explore the various designing possibilities using CSS. We were given the wireframe and asked to change the contents of the site as per our liking. I decided to go for a restaurant's website.

Creating this webpage made me realize of how unaware I was of designing concepts as well as different CSS that I could apply to make the UI better. It helped me get the desire to learn and improve myself on the same.

It may not be one of my finest but I include it in my portfolio as it is where I began my journey of learning front end development using HTML, CSS amd JavaScript.
